### 博客管理后台

Vite + Typescript + Vue3 + ElementPlus 开发的一个简易的博客管理后台，主要用于文章发布，评论管理，网站设置等等。

### 项目运行

1. 克隆项目到本地
2. 进入项目目录，安装依赖包 `npm install`
3. 运行项目 `npm run dev`
